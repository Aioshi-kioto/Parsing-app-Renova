
# Frontend - Zillow Parser UI

Vue 3 + Vite + Tailwind CSS фронтенд для парсинга Zillow.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Откроется на http://localhost:5173

## Сборка для production

```bash
npm run build
```

Собранные файлы будут в `dist/`

## Структура

- `src/App.vue` - Главный компонент
- `src/components/` - Компоненты UI
- `src/api.js` - API клиент
- `src/style.css` - Tailwind стили
